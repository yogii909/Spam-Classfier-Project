# 📧 Spam Classifier Project

A Machine Learning-based Spam Classifier that classifies text messages or emails as **Spam** or **Ham (Not Spam)** using Natural Language Processing (NLP) techniques.

## 🚀 Features

- Detects spam messages with high accuracy.
- Text preprocessing using NLP techniques.
- Machine Learning model for classification.
- User-friendly interface (if applicable).
- Fast and efficient predictions.

## 🛠️ Technologies Used

- Python
- Scikit-learn
- Pandas
- NumPy
- NLTK
- Streamlit / Flask (if used)
- Jupyter Notebook

```
Spam-Classfier-Project/
```

## 📊 Dataset

The project uses a spam classification dataset containing labeled messages as:
- Spam
- Ham (Not Spam)

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/ronak0408/Spam-Classfier-Project.git
```

Navigate to the project:

```bash
cd Spam-Classfier-Project
```

Install dependencies:

```bash
pip install -r requirements.txt
```

## ▶️ Run the Project

If using Streamlit:

```bash
streamlit run app.py
```

If using Flask:

```bash
python app.py
```
